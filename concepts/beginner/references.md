# Beginner References

Curated references for beginner-level PostgreSQL and SQL concepts.

See also: [root references.md](../../references.md) for the full curated list.

---

## Core references for this level

| Title | URL | Type | Level | Time | Why useful |
|-------|-----|------|-------|------|------------|
| PostgreSQL Official Tutorial | https://www.postgresql.org/docs/16/tutorial.html | Official Docs | Beginner | 2–3 h | Structured from-scratch intro written by the PostgreSQL team |
| PostgreSQL Tutorial (postgresqltutorial.com) | https://www.postgresqltutorial.com/ | Free Book | Beginner | 8–12 h | Topic-by-topic reference with runnable examples; covers SELECT through transactions |
| SQLZoo | https://sqlzoo.net/ | Blog | Beginner | 4–6 h | Browser-based interactive exercises; no local install needed |
| Select Star SQL | https://selectstarsql.com/ | Free Book | Beginner | 3 h | Interactive free book; teaches SQL through real executions dataset |
| W3Schools SQL Tutorial | https://www.w3schools.com/sql/ | Blog | Beginner | 3–5 h | Quick syntax reference with try-it editor; vendor-neutral |
| PostgreSQL Data Types | https://www.postgresql.org/docs/16/datatype.html | Official Docs | Beginner | 1 h | Every built-in type with storage size and valid ranges |
| psql meta-commands | https://www.postgresql.org/docs/16/app-psql.html | Official Docs | Beginner | 30 min | \d, \dt, \timing, \x — essential for interactive exploration |
| Connection Strings | https://www.postgresql.org/docs/16/libpq-connect.html | Official Docs | Beginner | 20 min | How DSN/connection strings work; environment variables |
| Mode Analytics SQL Tutorial | https://mode.com/sql-tutorial/ | Blog | Beginner | 4–6 h | Business-context SQL; covers GROUP BY, joins, aggregations clearly |
| pgtune (config calculator) | https://pgtune.leopard.in.ua/ | Blog | Beginner | 5 min | Generate sane postgresql.conf defaults based on hardware |
