# btree_gin (btree_gin)

Level: Intermediate
Available locally: Yes

## One-line purpose

Provide GIN operator classes for standard btree-comparable data types (int, text, timestamp, etc.), enabling composite GIN indexes that mix btree columns with full-text or hstore columns.
