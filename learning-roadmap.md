# Learning Roadmap

Full path from beginner to advanced PostgreSQL.

## Tracks

| Track | File | Focus |
|-------|------|-------|
| Beginner | `beginner-roadmap.md` | Intuition, setup, first queries, simple schema |
| Intermediate | `intermediate-roadmap.md` | Design, indexing, transactions, RLS, extensions |
| Advanced | `advanced-roadmap.md` | Internals, performance, agent-safe architecture |

## Capability coverage

Core SQL → Schema design → Constraints → Indexes → Query planning → Transactions → MVCC → Locks → Views → Functions → Triggers → Roles → RLS → Partitioning → Migrations → Observability

Non-SQL → JSONB → Full-text search → Fuzzy search → Vector search → Recursive CTEs → Graph patterns → Queue patterns → Audit/event-sourcing → Semantic memory

Extensions → See `extension-map.md`
